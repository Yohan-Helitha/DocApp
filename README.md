# AI-Enabled Smart Healthcare Appointment & Telemedicine Platform

This repository is scaffolded according to the SE3020 assignment specification.

## Architecture
- **Backend**: Microservices architecture
- **Frontend**: Monolithic layered architecture (asynchronous web client)
- **Infrastructure**: Docker + Kubernetes manifests

## Services (Backend)
1. auth-service
2. patient-management-service
3. doctor-management-service
4. appointment-service
5. telemedicine-service
6. payment-service
7. notification-service
8. admin-management-service
9. ai-symptom-checker-service (optional enhancement)

## Core folders
- `backend/` - microservices and API gateway
- `frontend/` - asynchronous web client (layered monolith)
- `infra/` - Docker and Kubernetes deployment artifacts
- `docs/` - microservice specifications and schema specifications
- `database/` - DB scripts and migrations placeholder
- `report/` - report artifacts placeholders
